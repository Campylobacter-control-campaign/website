# Campylobacter Control Campaign website starter

Static GitHub Pages starter website for the Campylobacter Control Campaign.

## Use

Copy these files into the GitHub repository, then enable GitHub Pages in **Settings → Pages → Deploy from branch → main / root**.

## Included updates

- CCC logo in `assets/CCClogo.png`
- current consortium member list in `consortium.html`
- GETCampy protocols.io laboratory and sequencing SOP links in `protocols.html`
